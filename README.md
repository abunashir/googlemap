# Googlemap

A simplest Ruby wrapper around Google Map API

## Installation

Add this line to your application's Gemfile:

```ruby
gem "googlemap", github: "abunashir/googlemap"
```

And then execute:

```
$ bundle install
```

## Usage

Don't! This just an experimental project for a specific purpose.
